

# CareerOS 🚀

### Level up from Graduate to Hired.

**Your AI Career Coach for South Africa.**

CareerOS is a modern AI-powered career accelerator designed specifically for South African graduates and early-career professionals. The platform combines artificial intelligence, gamification, and career development tools to help users secure jobs, internships, bursaries, and learnership opportunities while preparing them for success in the workplace.

Think of CareerOS as:

> **Duolingo + ChatGPT + LinkedIn for South African job seekers.**

---

# 📖 Project Overview

South Africa faces high levels of graduate unemployment, with many young professionals struggling to:

* Create professional job applications
* Prepare for interviews
* Find relevant opportunities
* Understand workplace communication
* Build confidence and career readiness

CareerOS addresses these challenges through an AI-first platform that acts as a personal career coach, helping users move from **Graduate → Hired**.

The application focuses on:

* Career development
* Job application assistance
* Interview preparation
* Opportunity discovery
* Workplace communication skills
* Professional growth through gamification

# ⚙️ Setup Instructions

## 1. Clone the Repository

```bash
git clone https://github.com/yourusername/careeros.git
cd careeros
```

---

## 2. Install Dependencies

```bash
npm install
```

---

## 3. Create Environment Variables

Create a `.env` file:

```env
VITE_OPENAI_API_KEY=your_openai_api_key
```

---

## 4. Start Development Server

```bash
npm run dev
```

The application will run at:

```txt
http://localhost:5173
```

---

## 5. Build for Production

```bash
npm run build
```

---

## 6. Preview Production Build

```bash
npm run preview
```
